BootStrap Notes

  • Twitter bootstrap is a front-end framework for UI and themes
  • Uses HTML/CSS/JS
  • It is mobile-first
  • Using a front-end framework increases development speed
  • Support from the community
  • It is customizable
  • Has navbar, labels and badges, tables, among more
  • Javascript Widgets such as Modals and transitions are available
  • Four different methods of installing: Manually, CDN, Bower, and NPM
  • Bower is a client-side package manager
  • NPM is also package manager, but server-side
  • JQuery needs to be included to be able to use all the widgets
