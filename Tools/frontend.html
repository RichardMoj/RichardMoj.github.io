Framework Selection

The framework that I have looked into is Material-UI (not to be confused with Material). I have always enjoyed Google's materialism standard and so I thought this would be a perfect framework to look into. Originally, I thought the Material theme was limited to only google, but I can see many other frameworks that try to follow this standard, including: Material, MUI, Material Components, among others. This framework tries to follow Google's standards on Material Design and it works very well with the Angular and React. Material-UI aims to be compatible among many various devices, while still keeping a clean and user-friendly UI. Additionally it's colors and themes are highly customizable coming with a dozen or sub-colors by default for the primary colors. and with the option to create your own. However, it does have it's downsides. First of all, it's documentation is just...good, not superb or anything. Thus, it is not meant to be used by inexperienced users. Additionally, It requires quite extensive knowledge in Node, and React. Furthermore, beyond it's color configuration and it's themes, it can't really be customized, at least not with difficulty as it wasn't designed with that in mind.
