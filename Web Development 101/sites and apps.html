Problems look mighty small from 150 miles up

Posted by Start Bootstrap on August 24, 2014

Contrary to popular belief, websites and webapps are two entirely different things. Websites are often built using only HTML/CSS and JavaScript. As a result, they are rather bare-bones looking. Web apps go one step further and employ the use of of a back-end language, for example python.

Websites

Websites are static and meant to just present information. They are only updated when the information found on that website is updated by a server administrator. It also accepts no user input such as login information or even comments. Such an example of a website is lela's pizzeria menu. As seen on the website, there is no interaction between the user and the website.

Web Apps

Web Apps are constantly updated and usually handles data. They are considered dynamic as the content presented to users is always changing. It often employs back-end tools to deal with the received data, such as name, password, comments. One of the biggest web apps is Twitch. People are constantly chatting in real time, or streaming videos. There are many tools that streamers can use to interact with their followers. These tools often accept input either through buttons or chat boxes.
