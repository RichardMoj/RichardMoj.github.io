Web Hosting

Free Web Hosting

The free web hosting option, Weebly, comes with 500MB of storage, unlimited bandwidth, and offers free website templates which are editable using HTML/CSS or a drag-n-drop option. They do not offer eCommerce (paid feature only), and email. It also uses a weebly.com subdomain.

Paid Web Hosting

The first paid web hosting service that came to mind was Amazon web Services. With AWS, you are able to use any content management system. It offers 3 categories of website hosting services: Simple website hosting, static website hosting, and enterprise web hosting.The pricing varies between all 3, with simple web hosting maximum pricing reaching $100/month for 8GB memory, 2 core processor, 80gb SSD diskspace and 5TB of transfer. Static web hosting was a bit more difficult to see the pricing of, but it seems to average about $0.30/GB. Enterprise web hosting prices ranged anywhere from $1,500 to $15,000 monthly.
