There is often a misunderstanding of what web development actually is. It is vital to know what web development actually consists of, and what it is to be expected from you as a web developer. Some view web development strictly as a web programming discipline, while others see it as more of an artistic creation of websites. Just like with any other area of study, web development contains many sub-disciplines. Web development can range from behind-the-scenes logical web programming, to visual design of the website, or the deployment of a website.

The sub-topics found under this section will help you understand the terms that are often brought up when discussing web development. Front-End/Back-End and Full-stacks are some of the most crucial information any web developer should be familiar. Additionally, it will go over the differences between the roles that developers participate in. The first step in this process is knowing how websites and webapps differ from each other.
